# Keep product execution capability independent from semantic feature activation

An asset/product capability release, account capability assignment, compliance/data/broker readiness,
reconciled state and kill generation jointly bound where an account may execute. A separate
DecisionFeatureActivation controls whether ontology-derived features may influence economics. Neither
authority implies the other, preventing a semantic promotion from enabling a product or a broker
integration from promoting unproven reasoning.
